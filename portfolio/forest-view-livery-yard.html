Forest View Livery Yard

A simple WordPress website for a Kent livery yard.

See Project

Re-design and development

One of my last projets before I left Suite Creative in 2015 was to transfer and re-design an old non-responsive site onto WordPress. Forest View was an already existing client of Suite Creative and it had gotten to the point where the clients existing website wasn’t suitable for purpose any more. As the client had requested the ability to edit the information on the website herself, WordPress was the ideal solution.

The color scheme remained the same for the project, but the layout and structure was all completely new. The site was updated with large beautiful imagery of the horses and also a much needed contact form for customers to get in contact with. The client was very happy with the website and agreed it was a much needed refresh.

SEO Updates

As an additional step to this project, some light SEO work was also carried out using the SEO WordPress plugin Yoast SEO. The plugin allowed me to fine tune meta descriptions and other ranking factors so that I was able to rank #1 for the term “livery yard east grinstead” which was great.
